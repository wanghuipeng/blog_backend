##### 1. 开发环境
 ```
node.js 7.6+
mognodb 3.0+
```
##### 2. 依赖于mongodb数据库
       先安装 [mongodb](https://www.mongodb.com/) 数据库，安装完成后运行数据库，开启27017（默认）端口
##### 3. 克隆到本地，安装依赖，运行
```
> git clone https://github.com/FantasyGao/blog-backend.git
> cd blog-backend
> npm install 
> npm start
```
